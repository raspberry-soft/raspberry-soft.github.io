Which tier should I get?
If you want to play the latest update as soon as it's made, you should get the 'MEGA FAN' tier. If you also want previews and behind the scenes, you should get the 'VIP' tier.

How frequently do you update?
Monthly updates. Each update consists of more story and a minimum of 2 new hand-drawn 2D CG images. You can always check the Changelog.

At which hour of the day the update is posted?
Usually around 21h00-23h00 Pacific Time. It may vary due to connectivity issues, last minute bug fixes and unforeseen events.

What's the difference between Public and Patreon builds?
Patreon builds contains the latest content regarding the story, always including new CG images. You need a password to unlock it. After 4 weeks it becomes public (Free) and you can find it on itch.io

What's on the Extras/Gallery Patch?
You'll get access to:

  • The 'FLOW' Button / 'Scene Selector' screen lets you jump to any point of the story in-game.
The latest game version works with the latest patch version.

These images represent the protagonist imagination, as part of the fetish is 'not knowing'. They will be unlocked after playing Hana POV, however, that part is NOT made yet.

What's the difference between the Gallery Patch and the CG Pack?
The patch lets you access an in-game image & scene gallery, and the pack contains all the CG image files for you to download in HD (1080p) & game (600p) resolution.
